公司简介

国泰君安证裕投资有限公司是国泰海通证券股份有限公司（证券代码：601211.SH，2611.HK；简称"国泰海通"）的全资子公司， 依照相关法律法规及监管要求，从事股权投资、金融产品投资，以及经许可的其他投资业务。

证裕投资依托国泰海通品牌优势及雄厚的资本实力、领先的投研水平，运用自有资金开展投资， 构建买方业务体系，提升国泰海通综合金融服务能力。

业务简介

证裕投资以股权投资、私募股权基金投资、科创板跟投为主要业务，秉持以"服务集团战略、推进组织协同、 改善资本收益"的战略定位，是国泰海通自有资金投资的重要实施平台， 落实集团长期资产配置和战略布局，实施长期价值投资。

核心业务板块

股权投资
基金投资
科创板跟投

公司依托国泰海通的品牌优势、客户资源和投研能力，聚焦国家战略性新兴产业， 重点投资具有核心技术和高成长性的优质企业，为股东创造长期稳定的投资回报。

经营理念

合规经营

严格遵守法律法规和监管要求，建立完善的内部控制体系， 确保各项业务合规、稳健运行。

专业投资

凭借专业的投资团队和丰富的投资经验，深入研究行业趋势， 精选优质投资标的。

诚信合作

坚持诚信为本，与投资伙伴建立长期稳定的合作关系， 实现互利共赢。

价值创造

以价值投资为导向，通过资本运作和资源整合， 为被投企业和股东创造价值。

投资理念

长期价值投资

坚持长期投资理念，关注企业的核心竞争力和长期发展潜力， 不追求短期投机收益，陪伴优质企业共同成长。

行业聚焦

聚焦国家战略性新兴产业，重点投资集成电路、生物医药、 人工智能、新能源等领域的优质企业。

价值发现

通过深入的行业研究和企业分析，发现被低估的优质投资标的， 获取长期投资价值。

投后赋能

不仅提供资金支持，更通过资源整合、管理提升等方式， 为被投企业提供全方位的投后服务。

投资领域

集成电路

投资半导体设计、制造、封装测试、设备材料等产业链关键环节， 支持国内集成电路产业发展。

芯片设计 晶圆制造 封装测试
